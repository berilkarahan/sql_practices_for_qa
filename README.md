# SQL QA Project – Portfolio Showcase

A hands-on SQL project built to demonstrate advanced data querying skills essential for QA engineers.

## Key Concepts

- Complex SQL Queries – design queries for multi-step data retrieval.
- GROUP BY & Aggregate Functions – SUM, COUNT, AVG, MAX, MIN.
- Subqueries – nested queries for advanced filtering and analysis.
- SQL Server Functions – string, date, and numeric operations.
- HAVING Clause – filter grouped results beyond WHERE.
- JOINs:
  - INNER JOIN – match records between tables
  - LEFT/RIGHT OUTER JOIN – preserve unmatched rows from one table
  - FULL OUTER JOIN – combine all records highlighting unmatched data

## Why This Project Matters

This project shows how a QA engineer can:

- Ensure data integrity across multiple tables
- Detect inconsistencies or anomalies
- Build reports and insights from complex datasets
- Demonstrate strong SQL and data validation skills
